Gitee Pages自动部署

原因：gitee pages页面不会自动更新，手动登录更新太过繁琐
解决办法：定时运行python脚本模拟浏览器登录gitee更新部署
具体办法：可以在本地制定定时任务或者云服务器设置定时任务
本文主要是在Ubuntu20.04.1 LTS阿里云轻量应用服务器上进行

安装python以及下载对应的运行库

1.检查系统是否安装了python

#在终端输入命令行
python  或  python3 (都尝试一下)

没有安装则会提示找不到命令Command 'python' not found
则进行下一步
2.安装依赖库

sudo apt-get install build-essential checkinstall

sudo apt-get install libreadline-gplv2-dev 

libncursesw5-dev libssl-dev libsqlite3-dev tk-dev 

libgdbm-dev libc6-dev libbz2-dev

3.创建文件夹

mkdir -p /etc/python3.6

4.进入文件夹

cd /etc/python3.6

5.下载安装包

wget https://www.python.org/ftp/python/3.6.6/Python-3.6.6.tgz

6.解压文件

sudo tar xzvf Python-3.6.6.tgz

7.编译python源

cd Python-3.6.6

sudo ./configure --enable-optimizations --
prefix=/etc/python3.6

sudo make altinstall

8.建立python3.6的软链接

ln -s /etc/python3.6/bin/python3.6 /usr/bin/python3.6

ln -s /etc/python3.6/bin/pip3.6 /usr/bin/pip3.6

9.将/etc/python3.6/bin加入PATH

vim ~/.bash_profile

复制以下代码,进入指定文件后，键盘按下 i 进入 insert 模式，添加如下内容，然后键盘按下ESC ,再输入 :wq,回车

10.激活配置

source ~/.bash_profile

11.检查安装

  • 查看python版本

    python3.6     # 输入pyhton3.6后回车即可

安装好后显示

  • 检查pip3是否存在

    pip3 -V   # V是大写的

    要是没有安装pip3

    sudo apt install python3-pip

12.使用pip3安装selenium库

pip3 install selenium

安装chrome浏览器

1.下载安装包

wget https://dl.google.com/linux/direct/google-chrome-stable_current_amd64.deb  #可能有点慢

2.安装chrome

sudo apt install ./google-chrome-stable_current_amd64.deb

3.查看是否安装成功

google-chrome --version

下载对应版本的chromedriver浏览器驱动

chromedriver下载地址

根据chrome浏览器版本下载对应的chromedriver版本（没有一样的，就用chromedriver版本低于chrome版本）

1.下载完本地解压或者云端解压

2.把解压出来的chromedriver放到/usr/local/share目录下

3.建立两条软链接

软链接类似于快捷方式，旨在不移动文件本地还可以使用

sudo ln -s /usr/local/share/chromedriver /usr/local/bin/   #在/usr/local/bin/目录下建立chromedriver的软链接
sudo ln -s /usr/local/share/chromedriver /usr/bin/			# /usr/bin/目录下建立chromedriver的软链接

4.赋予可执行权限

sudo chmod +x  /usr/local/share/chromedriver

编写自动部署python脚本及本地调试

我是在本地把代码调试好才上传到云服务器，本地是windows，运行同样需要python环境、谷歌浏览器、chromedriver


import time
from selenium import webdriver
from selenium.webdriver.common.alert import Alert

#定义选项（开启无头模式）
options = webdriver.ChromeOptions()
options.add_argument('--headless')
options.add_argument('--no-sandbox')
options.add_argument('--disable-gpu')
options.add_argument('--disable-dev-shm-usage')

# 打开到gitee登录界面
driver = webdriver.Chrome(options=options)

driver.get('https://gitee.com/login')


# 输入账号
user_login = driver.find_element('id','user_login')
user_login.send_keys("在这里输入账号 ")

# 输入密码
driver.find_element('id','user_password').send_keys("在这里输入密码")

# 点击登录按钮
driver.find_element('xpath',
    '//input[@class="ui fluid orange submit button large"]').click()
time.sleep(2)

# 切换到gitee pages界面--改为you_gitee_id
driver.get('https://gitee.com/你的账户名/你的仓库名/pages')


# 点击更新按钮--通过xpath确定点击位置
driver.find_element('xpath', '//div[@class="button orange redeploy-button ui update_deploy"]').click()

time.sleep(2)
# 确认更新提示框--这个函数的作用是确认提示框
Alert(driver).accept()

# 等待5秒更新
time.sleep(5)

# 这个print其实没事什么用,如果真的要测试脚本是否运行成功，可以用try来抛出异常
print("成功")

# 脚本运行成功,退出浏览器
driver.quit()

# 写上更新日志
# 我这里是写在T盘，可以改为自己喜欢的目录
fp = open("T:\工作台\gitee-deploy.txt", "a+")
now_time = time.strftime("%Y-%m-%d %H:%M:%S", time.localtime())
fp.write("部署时间:{0}\n".format(now_time))
fp.close()
