列表（list）

>>> python = ['list','tuple','set']     #新建一个列表
>>> python								#查看列表元素
['list', 'tuple', 'set']

增加元素

  • append

    使用方法

    list.append(obj)，obj是一个任意对象

>>> python.append('dict')			#在末尾添加元素
>>> python
['list', 'tuple', 'set', 'dict']
>>> python.append(10)		   #可以存储数字和字符串
>>> python
  ['list', 'tuple', 'set', 'dict', 10]
  
  #插入一个列表，作为一个元素插入
>>> type = ['列表','元组','集合']
>>> python.append(type)
>>> python
['list', 'tuple', 'set', ['列表', '元组', '集合']]

  • extend

    list.extend(seq)，seq是一个可迭代对象（列表，元组，字符串等）

  #插入一个列表，可以迭代插入
>>> type = ['列表','元组','集合']
>>> python.extend(type)
>>> python
['list', 'tuple', 'set', '列表', '元组', '集合']
  
  #插入一个字符串，会一个一个字符插入
>>> python
['list', 'tuple', 'set']
>>> python.extend('list')
>>> python
['list', 'tuple', 'set', 'l', 'i', 's', 't']
  • insert
    list.insert(index, obj)，index是对象obj需要插入的索引位置，obj是要插入列表中的对象
    >>> python
['list']
>>> python.insert(1,'tuple')
>>> python
['list', 'tuple']

删除元素

  • del

    del list[i]，i是列表元素的索引，删除元素

>>> python
['list', 'tuple', 'set', ['列表', '元组', '集合']]
>>> del python[3]
>>> python
['list', 'tuple', 'set']

  • pop

    list.pop(i)，i是列表元素的索引，弹出指定位置元素，没有指定则是最后一个元素

>>> python
['list', 'tuple', 'set', '列表', '元组', '集合']
>>> python.pop(3)
'列表'
>>> python
['list', 'tuple', 'set', '元组', '集合']

  • remove

    list.remove(value)，value是指定元素值

>>> python
['list', 'tuple', 'set', '元组', '集合']
>>> python.remove('元组')
>>> python
['list', 'tuple', 'set', '集合']

当列表有重复元素，默认移除第一个

>>> python
['list', 'tuple', 'set', 'list']
>>> python.remove('list')
>>> python
['tuple', 'set', 'list']

  • 切片

    object[start_index : end_index : step]

 >>> python
['list', 'tuple', '列表', '元组', '集合']
#删除‘列表’，[:2]从头到第2个，[3:]从第3个到结尾
>>> python = python[:2]+python[3:]
>>> python
['list', 'tuple', '元组', '集合']
切片操作

修改元素

  • list[i] = value
>>> python 
['list','tuple','set']
>>> python[2] =  '列表'
>>> python
['list', 'tuple', '列表']

查询元素

  • print(list[i])
>>> python
['list', 'tuple', '列表']
>>> print(python[2])

元组(tuple)

元组跟列表非常相似，用（）来表示，但是元组是不可变的，不能修改元组，元组可以迭代

>>> python = ('list','tuple','dict')  #新建元组，并初始化
>>> python			#查询全部
('list', 'tuple', 'dict')
>>> python[1]		#查询指定元素
'tuple'
>>> python.count('dict')		#查询元素出现次数
1
>>> python.index('list')		#查询元素的序号
0

#如果有两个相同元素，默认输出第一个的元素的序号
>>> python = ('list','tuple','dict','list')
>>> python.index('list')
0

>>> python = ('list')    #字符串作为迭代元素初始化元组
>>> python
'list'
>>> python[0]
'l'
>>> python[1]
'i'

>>> python = ('list',)     #创建只有一个元素的元组
>>> python[0]
'list'

>>> t = ['list','tuple','set']   #列表迭代初始化
>>> python = (t)
>>> python
['list', 'tuple', 'set']

字典(dict)

  • 字典是通过键值对的方式就数据存储下来，键必须是唯一的
  • 键值对在字典中以这样的方式标记：d = {key1 : value1, key2 : value2 }
  • 注意它们的键/值对用冒号分割，而各个对用逗号分割，所有这些都包括在花括号中
  • 另外，记住字典中的键/值对是没有顺序的
    #字典的操作，添加，删除，打印
>>> python = {'list':'列表','tuple':'元组','dict':'字典'}
>>> python['list']
'列表'

>>> python.values()		#查看值
dict_values(['列表', '元组', '字典'])

>>> python.keys()			#查看键
dict_keys(['list', 'tuple', 'dict'])

>>> python.items()       #查看键值对
dict_items([('list', '列表'), ('tuple', '元组'), ('dict', '字典')])

集合(set)

与字典类似，但只包含键，而没有对应的值，迭代初始化，包含的数据不重复
s = set(list or tuple or string)

#以list数据对集合初始化
>>> t = ['list','tuple','dict']
>>> python = set(t)
>>> python
{'list', 'tuple', 'dict'}
